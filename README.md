# ESP32-Blink-LED
30-Pin ESP32 ile harici LED yakıp söndürme (Blink) uygulaması.-- LED Blink application using a 30-Pin ESP32 development board.

🚀 ESP32 LED Control Project

Bu depo, 30-pinli ESP32 geliştirme kartı ile harici bir LED'in kontrol edilmesini içeren temel bir IoT giriş projesidir. Proje hem devre şemasını hem de optimize edilmiş kontrol yazılımını içerir.

This repository features a fundamental IoT starter project for controlling an external LED using a 30-pin ESP32 board. It includes both the circuit design and optimized firmware.

🔌 Donanım Yapılandırması / Hardware Configuration

Devre, güvenli akım sınırlaması ve düşük güç tüketimi odaklı tasarlanmıştır.

The circuit is designed with a focus on safe current limiting and low power consumption.
